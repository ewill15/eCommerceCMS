Texto de la pregunta

¿Qué código de respuesta se obtiene al ejecutar el siguiente código? (Reemplazando USER y PASSWORD por valores válidos):

$.ajax({
	url: ‘https://api.github.com/authorizations’,
	headers: {
		Authorization: ‘Basic’ +btoa(‘<USER>:<PASSWORD>’)
	},
	type: ‘POST’,
	data: {
		scopes:[“repo”],
		note: ‘Crear repositorio’
	},
	Complete: function(xhr,message,settings){
		console.log(xhr.status)
	}
})};

Retroalimentación

La respuesta correcta es: 400
