¿Qué diferencia existe entre la función require e include?

Seleccione una:

Retroalimentación

La función require indica que el archivo enlazado es estrictamente necesario para la ejecución del script, include también los enlaza pero de manera más flexible.

La respuesta correcta es: Con require, el script no se termina de ejecutar y muestra un error si el archivo enlazado no existe, mientras que con include no se mostraría error y se ejecutaría todo el script.
